Bienvenue à votre deuxième tutoriel!

Dans ce tutoriel, vous apprendrez les étapes nécessaires à un projet d’apprentissage automatique, de la définition de tâche à l'évaluation d'un modèle. Tout d'abord, nous présenterons quelques fonctionnalités de base de PyTorch, qui seront utilisées dans tous les tutoriels. Ensuite, nous allons construire notre pipeline de données, qui inclut un certain prétraitement pour obtenir le bon encodage pour les données catégorielles. Ensuite, vous définirez votre premier modèle : le fameux perceptron multicouche. Nous vous fournissons également la boucle d'entraînement, qui implémente la construction de mini-lots, la propagation avant, l'évaluation, la rétropropagation, et l’attribution de points de contrôle. Enfin, vous apprendrez à analyser les résultats de l’entraînement et à évaluer le modèle sur l'ensemble d’évaluation. Ce tutoriel dure environ 3-4 heures.
