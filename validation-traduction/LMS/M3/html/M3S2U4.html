Jeremy passe en revue quelques ensembles de données source ouverte classiques utilisés pour évaluer les performances d'une architecture de RNC dans diverses tâches, comme MNIST, fashion MNIST, SVHN, ImageNet et COCO. Il présente également quelques conseils importants à garder à l'esprit lors de la collecte de vos propres données pour les tâches d'apprentissage automatique. Les informations contenues dans cette vidéo sont utiles à tous ceux qui prévoient d'utiliser des RNC, que ce soit pour évaluer les performances d'un modèle ou pour exploiter vos propres données via un RNC.
