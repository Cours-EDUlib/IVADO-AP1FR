Golnoosh introduit l'idée de contraintes d'équité dans le problème de l'optimisation, ainsi que les difficultés potentielles découlant de ces contraintes supplémentaires. Elle présente aussi deux exemples. Le premier exemple, celui de l'équité en classification naïve bayésienne (2:30-3:18), introduit des concepts dépassant le niveau de ce cours et seule une compréhension superficielle est nécessaire. Le second exemple introduit l'équité dans les domaines relationnels. Golnoosh conclut en affirmant que chaque problème nécessite une solution détaillée pour inclure l'équité dans le processus d'optimisation.
